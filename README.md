# Fractal Explorer

An interactive fractal visualization tool with zoom, pan, color customization, and multiple fractal types.

## Features

- **Fractals**: Mandelbrot, Julia, Burning Ship, Tricorn
- **Zoom & Pan**: Mouse wheel to zoom, drag to pan
- **Colors**: Multiple palettes (Ocean, Cosmic, Fire, Aurora, Crystal, Plasma), background color picker
- **Parameters**: Max iterations, escape radius, Julia c constant

## Quick Start

1. Open `index.html` in a modern browser (Chrome, Firefox, Edge)
2. Or run a local server: `npx serve .` then visit `http://localhost:3000`
